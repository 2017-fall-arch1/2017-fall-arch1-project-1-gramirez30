# arch1-Binary-Search-Tree-demo

This directory contains:
* code that implements a Binary Search Tree of strings 
* a demo program that uses it

This program reads names, puts them in a BST and writes them to a file

This demo contains the following files:
 bstree.h: header file of BST structure & "public" interface functions
 bstree.c: implementation of Binary Search Tree
 bstDemo.c: a demonstration program that uses the Binary Search Tree
 

To compile:
~~~
$ make
~~~

To test it, try:
~~~
$ make demo
~~~

To delete binaries:
~~~
$ make clean
~~~
